# Installation

## Install egxpy

```bash
pip install git+https://github.com/egxlytics/egxpy.git
```

## Required Dependencies

- pandas
- numpy
- datetime
- holidays
- retry
- tvdatafeed (from `git+https://github.com/rongardF/tvdatafeed.git`)

Install all dependencies via requirements file:

```bash
pip install -r requirements.txt
```

## System Requirements

Python 3.7 or higher is required.
